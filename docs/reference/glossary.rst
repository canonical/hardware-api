.. _glossary:

Glossary
========

.. glossary::

    configuration
        A "configuration" is roughly equivalent to a
        :abbr:`SKU (Stock Keeping Unit)`. In most cases, the Ubuntu Certified
        status is assigned at the configuration level.

    platform
        Roughly speaking, a "platform" is a grouping of adjacent
        :abbr:`SKUs (Stock Keeping Units)`. In some cases, the Ubuntu Certified
        status is assigned at the platform level.
